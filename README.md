# Frozen Throne Frontend Audit & Refactor

## Project Description
The Frozen Throne Frontend Audit & Refactor project focuses on analyzing and improving a broken frontend application. The goal of the project is to identify structural, semantic, styling, and organizational issues in an existing codebase, then refactor it using proper HTML, CSS, and GitHub best practices. The project emphasizes understanding code quality, accessibility, responsiveness, and collaborative version control rather than speed.

## Features
- Semantic HTML structure using header, main, section, and footer
- Accessible login and signup forms with proper labels and input types
- Responsive layout using Flexbox
- Organized file structure with dedicated assets and images folders
- Clean and consistent file naming conventions
- Local image assets sourced from free stock image platforms
- GitHub Pages deployment for live preview
